# sitelock.tpl
Информация о том, что сайт временно недоступен.
Поддерживаемые переменные:
* {{ lock_reason }} - причина блокировки (устанавливается в админ панели)

Общие переменные для шаблонов:
* {{ what }} - ID NGCMS
* {{ version }} - версия NGCMS
