# Требования:
1. php версия не ниже 7.4
2. MySQL 5.6.15+

# Установка
1. Скачать содержимое данного репозитория в корневой каталог WEB сервера.
2. Открыть в WEB сервере ваш будущий сайт и следовать по указанным шагам.
